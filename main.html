Venueé

What's Venueé?

Venueé is an mobile event finding app that allows you to search for event in your location with the aid of GPS and a built in map in the application.

It allows businesses, promoters and event organisers to list their events along with descriptions, updates etc.

Guests can easily navigate and select events of interest, There is also the option of searching for events by name or by using tags.
